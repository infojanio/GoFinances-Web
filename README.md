# GoFinances-Web

Aplicação listando transações da API, frontend conectado ao backend, com upload de arquivo csv

🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

✔️Typescript

✔️React

✔️React Hooks

✔️Styled-components

✔️Polished

✔️Docker

✔️PostgreSQL

✔️Node

✔️Axios
